JavaScript for web development

Javascript is a programming language for the web and today forms a large part of the web stack. This course is basic and focuses on the client side (browser) and how Javascript can be used to make web pages interactive and dynamic and improve their functionality. The course is suitable for those who have no previous experience in programming or web development. Even you who can program and are interested in learning a new programming language in a new context can enjoy this course.
